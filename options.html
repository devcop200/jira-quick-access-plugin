Jira Quick Access — Settings

Connection

The base URL of your Jira instance (no trailing slash).

Generate one at Jira → Profile → Personal Access Tokens. Stored locally in your browser only.

✓ Saved

Returned Tickets

Issues matching this JQL are polled every minute in the background. Matching tickets appear in the Notifications tab and you get a desktop notification when the count increases.

Leave empty to disable background monitoring.

✓ Saved

Create Issue Defaults

Set defaults used when opening the Create Issue form from the popup.

Pre-selected project when you click + in the popup. You can still change it per issue.

✓ Saved

Appearance

Controls the colour scheme of the popup and settings page.

Watching

When enabled, the Watching tab hides tickets you've already resolved.

Notifications

The extension polls Jira every minute in the background and can show desktop notifications when something needs your attention.

Fires when more tickets match your Returned Tickets JQL filter than last poll.

Fires when someone mentions you in a comment (e.g. "John Doe mentioned you in PROJ-123").

Fires when a ticket is assigned to you that wasn't in the previous poll.
